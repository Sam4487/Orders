Zerodha Order App
=================

1. Install dependencies:
   pip install -r requirements.txt

2. Create a .env file from the .env.example and add your API credentials.

3. Run the app:
   python main.py

4. Use the GUI to:
   - Login to Zerodha
   - Place orders (LIMIT, MARKET)
   - Check live Profit & Loss
   - Toggle dark mode
